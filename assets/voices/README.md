## Supported Formats
- MP3 (recommended)
- WAV
- M4A

## File Size
Keep voice samples short (3-5 seconds) and under 500KB for optimal app performance.

## Example Content
Each voice sample should say something like:
"Hello, this is [Voice Name]. I'll be handling your restaurant calls."
